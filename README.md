# To Do List

## Description

Simple front end Angular application for generating a to do list. Options to Add, Update, and Delete.

## Tabe of Contents

- [Usage](#usage)

## Built With

- Angular
- Typscript
- Node.js
- HTML
- CSS

## Usage

Clone the repository. Run npm innstall and then npm start. Navigate to 'http://localhost:4200/'
